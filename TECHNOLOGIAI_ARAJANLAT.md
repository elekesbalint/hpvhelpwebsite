Technológiai Stack

Frontend

- Next.js 14+ (App Router) + TypeScript
- React 18+
- Tailwind CSS 3+
- shadcn/ui vagy Radix UI
- React Hook Form + Zod
- Zustand vagy React Query (state management)

Backend szolgáltatások (Supabase)

- Supabase Auth (bejelentkezés, regisztráció, jogosultság)
- Supabase PostgreSQL (adatbázis)
- Row Level Security (RLS) policyk
- Supabase Storage (képek és fájlok)

Adatbázis és Storage

- PostgreSQL (Supabase managed)
- Supabase Storage
- Opcionális Redis cache (későbbi skálázáshoz)

Szolgáltatások

- Email: SendGrid, Mailgun vagy AWS SES
- Fizetés: Stripe vagy PayPal
- Monitoring: Sentry

Deployment

- Frontend: Vercel vagy Netlify
- Backend Services: Supabase Cloud
- CI/CD: GitHub Actions vagy GitLab CI

Fejlesztési Időkeret

Maximum 3 hét

A fejlesztés párhuzamos munkával, gyors iterációval történik:

1. hét: Projekt setup, adatbázis design, authentication rendszer, alapvető CRUD műveletek
2. hét: Core funkciók (termékek, kosár, checkout, fizetési integráció), admin panel alapok
3. hét: Admin panel befejezése, tesztelés, optimalizáció, dokumentáció, deployment

A fejlesztés során a frontend és a Supabase backend szolgáltatások párhuzamosan készülnek, így a határidő betartása mellett is biztosítható a minőségi megoldás.

Összefoglalás

Ez a technológiai stack egy modern, skálázható és karbantartható webshop rendszert biztosít, amely képes kezelni a mai e-kereskedelmi igényeket. A választott technológiák ipari szabványok, jó dokumentációval és aktív közösséggel rendelkeznek.

Ajánlott stack összefoglalva:
- Frontend: Next.js 14+ (App Router) + TypeScript + Tailwind CSS
- Admin Panel: Next.js admin routes vagy külön admin app
- Backend szolgáltatások: Supabase (Auth + PostgreSQL + Storage + RLS)
- Cache: Redis (opcionális)
- Deployment: Vercel (frontend) + Supabase Cloud
